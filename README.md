
# EV Fault Code Dataset

Public machine-readable exports for [evfaultcode.com](https://evfaultcode.com), covering OBD-II diagnostic trouble codes, EV-specific fault codes, and Tesla proprietary alert codes.

## Current Coverage

- 3512 unique OBD-II code pages
- 3519 OBD-II dataset rows, including brand/platform variants
- 8265 Tesla alert codes
- 78 Tesla systems
- Supported languages in JSON translation objects: en, de, fr, es, nl, pt, tr, no, ko, zh
- Last generated: 2026-05-12
